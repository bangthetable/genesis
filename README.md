# Genesis

Genesis will setup your Macbook from scratch

## How to install

```
curl -fsSL https://raw.githubusercontent.com/bangthetable/genesis/master/start | ./start
```

## What it installs

### Terminal Utilities
- vim
- git
- fzf
- the_silver_searcher
- zsh
- z
- dnsmasq
- OhMyZsh

### Applications
- Chrome
- Firefox
- Slack
- Iterm2
- Sublime Text 3
- Alfred 3
- Docker edge

## After installation
- Setup ssh
- Make changes to the gitconfig added
